Автоматизированный голосовой/текстовый ассистент для салонов красоты, способный вести осмысленный диалог, консультировать клиентов на основе базы знаний и автоматически записывать их на услуги через интеграцию с Google Calendar.

## 📖 О Проекте

Цель проекта — разгрузить администраторов салона от рутинных звонков и сообщений, предоставив клиентам возможность записываться 24/7. Ассистент имитирует живого человека, умеет обрабатывать возражения, уточнять детали услуги и проверять свободные слоты в реальном времени.

## ✨ Ключевой Функционал

- **💬 Осмысленный диалог:** Поддержание контекста беседы, естественная речь, умение перефразировать и уточнять запросы клиента.
- **🧠 База Знаний (RAG):** Ответы на вопросы о ценах, мастерах, длительности процедур и правилах салона формируются строго на основе предоставленных данных.
- **📅 Интеграция с Google Calendar:** 
  - Проверка доступности слотов.
  - Бронирование времени.
  - Отправка подтверждений записи.
- **🎯 Сценарии:** Обработка стандартных ситуаций (запись, отмена, перенос, вопрос о цене).

## ⚙️ Архитектура и Промпт-Инжиниринг

Проект построен на основе тщательной проработки системного промпта. Для достижения наилучших результатов использовалась модель **Qwen**.

### Структура Системного Промпта
1.  **Role Definition:** Четкое определение роли (вежливый администратор, эксперт в бьюти-сфере).
2.  **Constraints:** Запрет на выдумывание цен и услуг (Hallucination control).
3.  **Context:** Информация о салоне, адрес, режим работы.
4.  **Tools:** Описание функций для вызова API календаря.

> **Пример логики:**
> `Если клиент хочет записаться -> Проверить слот в Google Calendar -> Если свободно -> Подтвердить -> Если занято -> Предложить альтернативу.`

## 🚀 Как это работает (Workflow)

1.  **Входящий запрос:** Клиент пишет или звонит с вопросом о записи.
2.  **Анализ намерения:** ИИ определяет интент (запись, вопрос, отмена).
3.  **Поиск в БЗ:** Если вопрос по услугам, ИИ обращается к векторной базе знаний (подготовленной через Qwen).
4.  **Действие:** Если нужна запись, ИИ через Suvvy отправляет запрос в Google Calendar.
5.  **Ответ:** Клиент получает подтверждение с датой и временем.
